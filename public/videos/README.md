# Hero Video

## Required Video File

Place `hero-showcase.mp4` in this directory for the homepage hero section.

### Video Specifications

- **Filename**: `hero-showcase.mp4`
- **Duration**: 6-12 seconds (loop)
- **Resolution**: 1920x1080 (Full HD) or higher
- **Format**: MP4 (H.264 codec recommended)
- **Audio**: Muted (no audio needed)
- **File Size**: Aim for < 5MB for optimal loading

### Content Recommendations

The video should showcase:
- Workspace/desk scene with Graffiti Designs products
- Subtle, professional motion (e.g., slow pan, gentle zoom)
- Good lighting with soft, warm tones
- Products arranged aesthetically
- Clean, minimalist composition

### Fallback Behavior

If `hero-showcase.mp4` is not present or fails to load, the component will automatically fall back to the static background image (`/bg/wood-light.jpg`).

### Example Sources

You can create or source videos from:
- Stock video sites (Pexels, Pixabay, Unsplash)
- Custom footage of your products
- Motion graphics created in After Effects/Premiere Pro
